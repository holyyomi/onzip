# 온집 실기기 QA

> 목적: 운영 배포본이 실제 휴대폰에서 앱처럼 자연스럽게 설치되고 열리는지 확인한다.

## 대상

- 운영 URL: `https://onzip.vercel.app`
- Android Chrome
- iPhone Safari
- 이미 설치한 PWA가 있다면 삭제하지 말고 기존 데이터 유지 여부를 먼저 확인한다.

## 공통 확인

- [ ] 첫 로딩이 2초 안팎으로 끝난다.
- [ ] 홈 첫 화면에서 `오늘의 주요 항목`, 빠른 추가, `생활 할 일`, `이번 달 흐름`이 보인다.
- [ ] 하단 탭이 손가락으로 누르기 쉽고 화면 하단에 잘리지 않는다.
- [ ] 하단 탭의 `생활`을 눌러 장보기와 체크리스트 화면으로 이동할 수 있다.
- [ ] 빠른 추가에서 `지출 예정`을 저장하면 `흐름` 탭으로 이동하고 목록이 바로 갱신된다.
- [ ] 빠른 추가에서 `중요 일정`을 저장하면 `일정` 탭으로 이동하고 캘린더에 바로 보인다.
- [ ] `보관 메모` 저장 후 `금고` 탭으로 이동하고 검색으로 찾을 수 있다.
- [ ] 설정에서 `백업 파일 내려받기`와 `백업 파일로 교체하기`가 서로 다른 행동으로 이해된다.
- [ ] 설정에서 `민감 메모 숨김`을 켜면 4자리 PIN 설정만 요구하고 앱 전체 잠금처럼 느껴지지 않는다.

## Android Chrome

- [ ] `https://onzip.vercel.app` 접속 후 설치 안내가 보인다.
- [ ] 설치 버튼 또는 Chrome 메뉴의 `홈 화면에 추가`로 설치할 수 있다.
- [ ] 홈 화면 아이콘이 온집 아이콘으로 보인다.
- [ ] 설치한 앱을 열면 브라우저 주소창 없이 단독 앱처럼 열린다.
- [ ] 설치 안내를 닫으면 바로 다시 나타나지 않는다.
- [ ] 4G/5G 또는 Wi-Fi 변경 후에도 앱이 열린다.

## iPhone Safari

- [ ] Safari에서 `https://onzip.vercel.app` 접속 후 수동 설치 안내가 이해된다.
- [ ] 공유 버튼의 `홈 화면에 추가`로 설치할 수 있다.
- [ ] 홈 화면 아이콘과 앱 이름이 자연스럽게 보인다.
- [ ] 설치한 앱을 열면 Safari 주소창 없이 열린다.
- [ ] 새로 저장한 데이터가 앱을 닫았다 다시 열어도 유지된다.

## 업데이트 확인

- [ ] 새 배포 이후 앱을 다시 열면 업데이트 안내가 보인다.
- [ ] 업데이트 적용 후 화면이 새로고침되고 데이터가 유지된다.
- [ ] 설정의 최신 업데이트 카드에서 현재 버전 확인/적용 흐름이 어색하지 않다.
