# 온집 실사용 테스트 체크리스트

## 실행 방법

```bash
npm run dev
# 브라우저에서 http://localhost:3000 열기
```

휴대폰에서 볼 때는 같은 Wi-Fi에서 `http://{PC IP}:3000`으로 접속한다.
운영 배포본은 `https://onzip.vercel.app`에서 확인한다.
설치와 실제 휴대폰 확인은 `docs/DEVICE_QA.md`를 함께 사용한다.

---

## 골든 패스

### 1. 홈에서 지출 예정 남기기
- [ ] 홈 탭 첫 화면에 `오늘의 주요 항목`, 빠른 추가, `이번 달 흐름`이 보인다.
- [ ] 홈 빠른 추가에서 `지출 예정`을 누른다.
- [ ] 금액과 분류만 입력해 저장한다.
- [ ] 저장 알림이 뜨고 `흐름` 탭으로 이동한다.
- [ ] 이번 달 흐름이나 기록 목록에 방금 입력한 항목이 반영된다.

### 2. 수입 예정 남기기
- [ ] 홈 또는 흐름 탭에서 `수입 예정`을 누른다.
- [ ] 큰 금액 추천 버튼이 수입 입력에 맞게 보인다.
- [ ] 금액 입력 후 Enter 또는 저장 버튼으로 저장한다.
- [ ] 저장 후 `흐름` 탭에서 남은 수입/예상 차액이 갱신된다.

### 3. 중요 일정 추가
- [ ] 일정 탭에서 `일정 넣기`를 누른다.
- [ ] 일정명과 날짜만 입력해 저장한다.
- [ ] 저장 후 `일정` 탭으로 이동한다.
- [ ] 월간/주간 캘린더와 선택 날짜 패널에 일정이 보인다.

### 4. 보관 메모 추가
- [ ] 금고 탭에서 `보관 메모 남기기`를 누른다.
- [ ] 템플릿 없이 제목 또는 내용만 입력해 저장한다.
- [ ] 저장 후 `금고` 탭에서 메모가 보인다.
- [ ] 검색어를 입력하면 제목/내용/태그 기준으로 찾을 수 있다.

### 5. 민감 메모 PIN
- [ ] 설정에서 `민감 메모 숨김`을 켠다.
- [ ] PIN이 없으면 `민감 메모 PIN` 영역이 열린다.
- [ ] 4자리 PIN을 설정한다.
- [ ] `민감` 태그가 붙은 보관 메모를 열 때만 PIN 확인이 나온다.
- [ ] 설정에서 새 4자리 PIN으로 바로 바꾸거나 PIN을 끌 수 있다.

### 6. 구매 항목 추가
- [ ] 빠른 추가 또는 생활 화면에서 `구매 항목`을 저장한다.
- [ ] 저장 후 숨겨진 생활 화면의 장보기 목록으로 이동한다.
- [ ] 체크 버튼을 누르면 구매 완료 상태로 이동한다.

### 7. 백업과 불러오기
- [ ] 설정의 `앱 링크 공유`가 데이터 공유가 아니라 설치 링크 공유로 이해된다.
- [ ] `백업 파일 내려받기`를 누르면 JSON 파일을 내려받는다.
- [ ] `백업 파일로 교체하기` 버튼명이 현재 데이터를 교체하는 동작으로 이해된다.
- [ ] 백업 파일 선택 후 확인창에 파일명과 교체 안내가 보인다.

## 기능별 확인

### 홈
- [ ] 오늘 미납/입금 대기/자동결제 확인 필요 항목이 일정보다 먼저 보인다.
- [ ] 오늘 항목이 5건을 넘으면 숨겨진 건수와 흐름 이동 버튼이 보인다.
- [ ] 빠른 추가 4개는 `지출 예정`, `수입 예정`, `중요 일정`, `보관 메모` 중심이다.
- [ ] 금액 가리기를 켜면 금액이 `***원`으로 보인다.

### 흐름
- [ ] 월 이동 버튼으로 기준 월이 바뀐다.
- [ ] 이번 달 예상 흐름, 남은 수입, 남은 지출, 예상 차액이 보인다.
- [ ] 날짜별 흐름에서 `오늘`, `D-n`, `지남`, `입금 대기`, `미납`, `확인필요` 배지가 자연스럽다.
- [ ] 수입은 `입금/취소`, 지출은 `완료/취소`, 자동결제는 `결제/취소` 처리할 수 있다.
- [ ] 정기 항목 탭에서 선택한 월의 처리 상태가 다른 달로 새지 않는다.

### 일정
- [ ] 한 달/한 주 전환이 정상 동작한다.
- [ ] 오늘 버튼이 오늘 날짜로 돌아간다.
- [ ] 정기 지출/자동결제 자동 일정은 월별 완료 상태를 반영한다.
- [ ] 일정 직접 추가/수정/삭제가 가능하다.

### 금고
- [ ] 유형 필터, `중요만`, `민감만` 필터가 정상 동작한다.
- [ ] 갱신/만료 메모는 날짜가 가까우면 홈의 다가오는 항목에 표시된다.
- [ ] 민감 메모 숨김을 켜면 제목/내용이 가려진다.
- [ ] PIN이 설정된 민감 메모는 열람 전 4자리 PIN을 요구한다.

### 생활
- [ ] 빠른 추가로 저장한 구매 항목/체크리스트가 생활 화면에 바로 반영된다.
- [ ] 체크리스트 항목 추가와 진행률이 정상 동작한다.
- [ ] 생활용품의 구매 필요 항목을 구매 항목으로 보낼 수 있다.
- [ ] 집안일 완료 상태를 바꿀 수 있다.

### 설정
- [ ] 집 이름 저장 후 헤더 이름이 즉시 바뀐다.
- [ ] 금액 가리기와 민감 메모 숨김이 즉시 반영된다.
- [ ] 설치 안내를 닫아도 7일 후 다시 볼 수 있다.
- [ ] 백업 상태 배지가 최근 백업 상태를 보여준다.

## 실패 신호

- [ ] 입력 전에 어느 버튼을 눌러야 할지 망설인다.
- [ ] 백업 불러오기를 데이터 합치기로 오해한다.
- [ ] 민감 메모 PIN이 앱 전체 잠금처럼 무겁게 느껴진다.
- [ ] 홈 첫 화면에서 핵심 정보가 보이지 않는다.
- [ ] 설치 안내를 놓치면 다시 찾기 어렵다.
