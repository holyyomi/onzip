# 온집 실사용 테스트 체크리스트

## 실행 방법

```bash
npm run dev
# 브라우저에서 http://localhost:3000 열기
```

휴대폰에서 볼 때는 같은 Wi-Fi에서 `http://{PC IP}:3000`으로 접속한다.
운영 배포본은 `https://onzip.vercel.app`에서 확인한다.
설치와 실제 휴대폰 확인은 `docs/DEVICE_QA.md`를 함께 사용한다.

---

## 골든 패스

### 1. 홈에서 지출 예정 남기기
- [ ] 홈 탭 첫 화면에 `오늘의 주요 항목`, 빠른 추가, `체크리스트`, `이번 달 가계부`가 보인다.
- [ ] 홈 빠른 추가에서 `지출 예정`을 누른다.
- [ ] 금액과 분류만 입력해 저장한다.
- [ ] 저장 알림이 뜨고 `가계부` 탭으로 이동한다.
- [ ] 이번 달 가계부나 상세내역에 방금 입력한 항목이 반영된다.

### 2. 수입 예정 남기기
- [ ] 홈 또는 가계부 탭에서 `수입`을 누른다.
- [ ] 큰 금액 추천 버튼이 수입 입력에 맞게 보인다.
- [ ] 금액 입력 후 Enter 또는 저장 버튼으로 저장한다.
- [ ] 저장 후 `가계부` 탭에서 남은 수입/예상 차액이 갱신된다.

### 3. 중요 일정 추가
- [ ] 일정 탭에서 `일정 추가`를 누른다.
- [ ] 일정명과 날짜만 입력해 저장한다.
- [ ] 저장 후 `일정` 탭으로 이동한다.
- [ ] 월간/주간 캘린더 날짜 칸과 선택 날짜 패널에 일정 제목이 보인다.

### 4. 메모 추가
- [ ] 메모장 탭에서 `메모 남기기`를 누른다.
- [ ] 템플릿 없이 제목 또는 내용만 입력해 저장한다.
- [ ] 저장 후 `메모장` 탭에서 메모가 보인다.
- [ ] 검색어를 입력하면 제목/내용/태그 기준으로 찾을 수 있다.

### 5. 메모장 비밀 표시
- [ ] 메모 내용 입력 오른쪽에 `공개`와 `비밀` 두 선택지가 함께 보인다.
- [ ] `비밀`을 선택한 메모는 목록에서 제목은 보이고 내용만 가려진다.
- [ ] 비밀 메모를 누르면 PIN 없이 바로 수정 화면이 열린다.

### 6. 체크리스트 추가
- [ ] 빠른 추가 또는 체크리스트 화면에서 체크리스트 항목을 저장한다.
- [ ] 저장 후 하단 `체크리스트` 탭으로 이동한다.
- [ ] 체크박스를 누르면 줄긋기 상태가 바로 반영된다.
- [ ] 항목을 수정하고 삭제할 수 있다.

### 7. 백업과 불러오기
- [ ] 설정의 `앱 링크 공유`가 데이터 공유가 아니라 설치 링크 공유로 이해된다.
- [ ] `백업 파일 내려받기`를 누르면 JSON 파일을 내려받는다.
- [ ] `백업 파일로 교체하기` 버튼명이 현재 데이터를 교체하는 동작으로 이해된다.
- [ ] 백업 파일 선택 후 확인창에 파일명과 교체 안내가 보인다.

## 기능별 확인

### 홈
- [ ] 오늘 미납/입금 대기/자동결제 확인 필요 항목이 일정보다 먼저 보인다.
- [ ] 오늘 항목이 5건을 넘으면 숨겨진 건수와 가계부 이동 버튼이 보인다.
- [ ] 홈 빠른 추가는 `지출 예정`, `수입 예정`, `체크리스트`, `중요 일정`, `메모장`이 보인다.
- [ ] `체크리스트`에서 미완료 체크리스트를 확인하고 체크리스트 탭으로 이동할 수 있다.
- [ ] 금액 가리기를 켜면 금액이 `***원`으로 보인다.

### 가계부
- [ ] 월 이동 버튼으로 기준 월이 바뀐다.
- [ ] 이번 달 가계부 요약, 남은 수입, 남은 지출, 예상 차액이 보인다.
- [ ] 날짜별 입출금에서 `오늘`, `D-n`, `지남`, `입금 대기`, `미납`, `확인필요` 배지가 자연스럽다.
- [ ] 수입은 `입금/취소`, 지출은 `완료/취소`, 자동결제는 `결제/취소` 처리할 수 있다.
- [ ] `입출금` 탭에서 선택한 월의 처리 상태가 다른 달로 새지 않는다.

### 일정
- [ ] 한 달/한 주 전환이 정상 동작한다.
- [ ] 오늘 버튼이 오늘 날짜로 돌아간다.
- [ ] 달력 날짜 아래에 일정 제목 미리보기가 보인다.
- [ ] 정기 지출/자동결제 자동 일정은 월별 완료 상태를 반영한다.
- [ ] 일정 직접 추가/수정/삭제가 가능하다.

### 메모장
- [ ] 검색창 아래 유형별 빠른 추가 버튼이 가로 스크롤 없이 여러 줄로 보인다.
- [ ] 갱신/만료 메모는 날짜가 가까우면 홈의 다가오는 항목에 표시된다.
- [ ] `비밀`을 선택한 메모는 별도 설정 없이 제목은 보이고 내용만 가려진다.
- [ ] 비밀 메모를 눌러도 PIN 없이 바로 열린다.

### 체크리스트
- [ ] 빠른 추가로 저장한 체크리스트가 체크리스트 화면에 바로 반영된다.
- [ ] 화면 상단 입력창에서 항목을 바로 추가할 수 있다.
- [ ] 체크박스 선택 시 줄긋기가 적용된다.
- [ ] 수정/삭제만 보여서 다른 생활 기능으로 헷갈리지 않는다.

### 설정
- [ ] 집 이름 저장 후 헤더 이름이 즉시 바뀐다.
- [ ] 금액 가리기가 즉시 반영된다.
- [ ] 설치 안내를 닫아도 7일 후 다시 볼 수 있다.
- [ ] 백업 상태 배지가 최근 백업 상태를 보여준다.

## 실패 신호

- [ ] 입력 전에 어느 버튼을 눌러야 할지 망설인다.
- [ ] 백업 불러오기를 데이터 합치기로 오해한다.
- [ ] 메모장 비밀 표시가 앱 전체 잠금처럼 무겁게 느껴진다.
- [ ] 홈 첫 화면에서 핵심 정보가 보이지 않는다.
- [ ] 설치 안내를 놓치면 다시 찾기 어렵다.
