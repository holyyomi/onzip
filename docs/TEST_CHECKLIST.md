# 온집 실사용 테스트 체크리스트 (TASK-031)

## 실행 방법
```bash
npm run dev
# 브라우저에서 localhost:3000 열기
# 모바일: 같은 Wi-Fi에서 http://{PC IP}:3000
```

---

## 골든 패스 시나리오 (순서대로 테스트)

### 시나리오 0. 홈 빠른 추가
- [ ] 홈 탭에서 "지출 기록" 클릭
- [ ] 금액/카테고리만 입력 → 저장
- [ ] "지출 기록이 저장되었습니다" 알림 확인
- [ ] 홈의 월간 요약 지출 금액이 새로고침 없이 반영되는지 확인
- [ ] "날짜/세부 정보 자세히"를 열면 추가 옵션이 보이는지 확인

### 시나리오 1. 오늘 중요 일정 추가
- [ ] 캘린더 탭 진입
- [ ] 오늘 날짜 탭 클릭 → DayEventPanel 표시
- [ ] "중요 일정" 클릭 → EventFormModal 열림
- [ ] 제목 입력 → 저장
- [ ] 캘린더에 파란 dot 표시 확인
- [ ] 오늘 요약 카드의 "일정" 카운트 +1 확인

### 시나리오 2. 정기 지출 추가
- [ ] 흐름 탭 → 정기 지출 서브탭
- [ ] "+ 추가" → 이름/금액/납부일 입력 → 저장
- [ ] 목록에 표시 확인
- [ ] 캘린더로 이동 → 해당 납부일에 빨간 dot 표시 확인
- [ ] 납부완료 체크 → 완료 상태로 변경 확인

### 시나리오 3. 지출 입력
- [ ] 돈관리 → 가계부 서브탭
- [ ] "+ 지출" → 금액/카테고리/결제수단 입력 → 저장
- [ ] 날짜 그룹으로 목록 표시 확인
- [ ] 요약 탭 → 이번 달 지출 합계 반영 확인

### 시나리오 4. 장보기 추가
- [ ] 생활 탭 → 장보기 서브탭
- [ ] "+ 추가" → 구매 항목명 입력 → 저장
- [ ] 예상 금액 합계 표시 확인
- [ ] 체크 버튼 클릭 → 구매완료 영역으로 이동 확인
- [ ] "자주 사는 품목" 탭 → 즐겨찾기 품목 확인

### 시나리오 5. 생활용품 상태 변경
- [ ] 생활 탭 → 생활용품 서브탭
- [ ] Seed 데이터 확인 (휴지, 세제 등)
- [ ] 상태 배지 탭 → 충분함→부족함→구매필요 순환 확인
- [ ] "구매필요" 항목 → "장보기↗" 버튼 클릭 → 장보기 추가 확인

### 시나리오 6. 가족 회의록 작성
- [ ] 홈 탭의 "가족 기록" 카드 클릭 → 기록 탭 진입
- [ ] "가족 회의록" 빠른 추가 버튼 클릭
- [ ] 회의록 템플릿 자동 삽입 확인
- [ ] 내용 수정 → 저장
- [ ] 태그 입력 → 태그 검색으로 찾기 확인

### 시나리오 7. 템플릿 복사
- [ ] 생활 탭 → 템플릿 서브탭
- [ ] "신혼집 첫 세팅 체크리스트" → 항목 미리보기
- [ ] "복사하기" 클릭 → 확인 alert
- [ ] 체크리스트 서브탭 → 새 체크리스트 생성 확인

## 기능별 체크리스트

### 캘린더
- [ ] 월 이동 (이전/다음) 정상 동작
- [ ] 오늘 날짜 파란 원으로 강조
- [ ] 이전 달 날짜 회색으로 표시
- [ ] 일요일 빨간색, 토요일 파란색
- [ ] 이벤트 dot 3개 초과 시 3개만 표시
- [ ] 다른 달 날짜 클릭 → 해당 달로 자동 이동

### 돈관리
- [ ] 요약: 수입-지출-정기 지출-구독 4개 카드 표시
- [ ] 남은 생활비 음수 시 빨간색 표시
- [ ] 가계부: 날짜 내림차순 그룹 표시
- [ ] 구독: "연 X원" 자동 계산 표시
- [ ] 계산기: 기본/분담/구독 3종 모드 전환

### 생활
- [ ] 체크리스트: 진행률 바 실시간 업데이트
- [ ] 체크리스트: 인라인 항목 추가 (Enter 키)
- [ ] 집안일: 완료 체크 → 완료 그룹으로 이동

### 기록
- [ ] 유형 필터 → 해당 유형만 표시
- [ ] 검색: 제목/내용/태그 포함 검색
- [ ] 날짜 내림차순 그룹 표시

### 설정
- [ ] 구성원 색상 변경 → 즉시 반영
- [ ] 기본 구성원(나/배우자/공동) 삭제 불가 확인
