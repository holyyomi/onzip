> 목적: 실제 가족 사용 기준으로 불편한 흐름을 빠르게 찾는다.

## 매일 확인

- 홈 첫 화면에서 오늘 할 일과 빠른 기록이 바로 보이는지 확인
- 지출 기록, 구매 항목, 중요 일정 등록을 10초 안에 끝낼 수 있는지 확인
- 하단 메모가 잘리지 않고 입력/수정되는지 확인
- 앱을 닫았다 다시 열어도 입력 데이터가 유지되는지 확인

## 3일 차 확인

- 생활 탭에서 체크리스트, 장보기, 생활용품, 집안일의 구분이 헷갈리지 않는지 확인
- 빈 화면에서 다음 행동이 자연스러운지 확인
- 설정의 백업 파일 내려받기와 불러오기가 이해되는지 확인
- 공유하기가 앱 링크 공유로 이해되고, 내 데이터 공유로 오해되지 않는지 확인

## 7일 차 확인

- 돈 탭에서 월간 지출/수입/고정 지출 흐름이 한눈에 들어오는지 확인
- 기록 탭에서 회의록, 생활 기록, 집 관련 메모를 다시 찾기 쉬운지 확인
- 설치한 PWA가 일반 앱처럼 실행되고 업데이트 후 다시 열었을 때 최신 화면이 보이는지 확인
- 사용자가 반드시 알아야 할 로컬 저장 한계가 과하게 거슬리지 않으면서도 충분히 전달되는지 확인

## 판단 기준

- 같은 기능을 두 번 이상 설명해야 하면 문구나 위치를 수정한다.
- 입력 전에 망설이는 화면은 버튼명이나 기본값을 수정한다.
- 홈에서 첫 화면을 내려야 핵심 정보가 보이면 카드 밀도를 다시 조정한다.
- 데이터가 사라질 수 있는 행동은 설정의 안내와 백업 흐름을 더 명확하게 만든다.

## 우선순위 분류

| 우선순위 | 기준 | 예시 |
|---|---|---|
| P0 | 데이터 손실, 저장 실패, 앱 실행 불가 | 백업 파일 교체 후 데이터가 깨짐 |
| P1 | 핵심 흐름을 끝내지 못함 | 지출 예정 저장 후 어디에 저장됐는지 모름 |
| P2 | 반복 사용이 불편함 | 매번 자세히를 열어야 해서 입력이 느림 |
| P3 | 문구나 시각적 어색함 | 버튼명은 이해되지만 더 짧게 바꿀 수 있음 |
